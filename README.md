# 😈 Unfair Hangman Game (C++)

A console-based Hangman game built in C++ — but with a twist...  
This game is intentionally unfair 😏

---

## 🎮 About the Game

This is not your typical Hangman.

The game is designed to:
- Challenge the player
- Punish mistakes harshly
- Reward unpredictably

⚠️ Warning: This game is meant to be frustrating (in a fun way)

---

## 🚀 Features

- 🎲 Random word selection
- 💡 Hint system for each word
- 🔤 Letter-by-letter guessing
- ❤️ Limited chances system
- 📉 Score system
- 😈 Unfair mechanics:
  - Score may increase multiple times for a single guess
  - Wrong guesses reduce both score and chances
  - Repeated guesses are blocked

---

## 🧠 Game Rules

- Guess the hidden word using the hint
- Enter only one letter at a time
- Correct letters will be revealed
- Wrong guesses reduce chances and score
- Win: Guess the word
- Lose: Run out of chances

---

## 🕹️ How to Play

1. Run the program
2. Read the hint
3. Start guessing letters
4. Try to survive the unfair system 😈
